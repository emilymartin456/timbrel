# 安全策略

## 支持的版本

本项目处于研究早期，仅对最新发布版本提供安全修复。

| 版本 | 是否支持 |
|---|---|
| 0.1.x | ✅ |
| < 0.1 | ❌ |

## 报告漏洞

如果你发现安全问题，请**不要**直接提交公开 issue。请通过 GitHub 的
「Report a vulnerability」（私有漏洞报告）渠道提交，我们会尽快跟进，并在修复
发布后再公开细节。

由于本项目主要涉及模型与音频处理，请特别留意：

- 反序列化不受信任的 checkpoint（`torch.load`）带来的风险；
- 处理外部音频/文本输入时的资源耗尽问题。

感谢你负责任地披露问题。
