# 贡献者行为准则

## 我们的承诺

为营造开放、友好的环境，我们承诺：无论年龄、体型、身体健全与否、族裔、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向如何，参与本项目的每个人都能获得免于骚扰的体验。

## 我们的准则

有助于营造积极环境的行为包括：

- 使用友善、包容的语言；
- 尊重不同的观点与经验；
- 平和地接受建设性批评；
- 以社区整体利益为重。

不可接受的行为包括：

- 带有性暗示的言语或图像，以及任何形式的性骚扰；
- 挑衅、侮辱或人身攻击；
- 公开或私下的骚扰；
- 未经许可公布他人隐私信息。

## 责任与范围

项目维护者有权删除、编辑或拒绝不符合本准则的评论、提交、代码与其他贡献。本准则适用于项目空间内以及在公共场合代表项目的场景。

## 执行

如遇违规行为，可通过 GitHub 私有漏洞/举报渠道联系维护者。所有投诉都会被审阅与调查，并作出适当回应。

本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) v2.1。
